### 🧠 What I Do

I sit at the intersection of **software engineering** and **applied AI**, building systems that go beyond prototypes:

- 🏗️ **Multi-agent architectures** — orchestration pipelines using LangChain, vector search (pgvector), and queue-based workers (BullMQ, Redis)
- ⚙️ **Workflow automation at scale** — n8n and Make.com systems integrating CRMs, payment flows, and third-party APIs for real operational throughput
- 🌐 **Full-stack product delivery** — Next.js, Supabase, and PostgreSQL applications from schema design to deployed UI
- 🤖 **Voice & conversational AI** — Vapi.ai-based voice agents and GoHighLevel CRM integrations
- 🕸️ **Data pipelines** — Playwright-based scraping and structured data extraction feeding into automation and BI systems

I've delivered systems across **healthcare, real estate, compliance, and professional services** — environments where automations need to be reliable, auditable, and genuinely reduce human workload, not just demo well.
